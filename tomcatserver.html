Project 2: Tomcat Server Website

Description

This project was to design a website which allows for certain users to execute commands using SQL on an Apache Tomcat Server.
There were two different types of users in this project. The Root user and the Client user. Each type had different
permissions regarding the database. The root user could add and subtract while the user could only see items within
the database.

Skills

-Creation of a website -Basic UI design -Implementation of permissions per user type -How to use SQL database commands to add, subtract, etc.
