Project 1: Java Shop GUI

Description

This project was designed to create a basic shop GUI for users to enter the items they wished to purchase.
The program loads a file which contains all itmes available which allows users to add them to a cart if
they are available. Invalid or unavailable items will not be added to the cart. Once the user clicks
the checkout button, a recipt document is output detailing what the user bought, how many they bought, and
the total cost of the transaction.
